QueryBuilder
showColumns – метод для отображения колонок таблицы
getAll – Вывод всех дынных с запрашиваемой таблицы
getOne - Вывод данных по id
create - создание записи в БД
updata - обновление данных в БД по ид
delete - удаление данных по ид
find - найти в колонке по искомому слову
getByColumn - Вывод данных по уазанным значениям

Router
В Маршрутизаторе описан каждый метод.
В фронтконтроллере создается правило с регулярными выражениями,
где по умолчанию, если URI пустой контроллер Мейн, и экшен Индекс,
а если что-то есть то данные додоются в метод add() который в свою
очередь записывает их в массив routes. Метод matchRoute записывает
conteroller и action, если action нет то создает по умолчанию index.
dispatch проверяет есть или доступен ли класс и метод запрашиваемые
в а адресной строке, если ксть то подключает, если нет то выводит 404ю

В адресной строке можна вписать и запрос с тере, метод upperCamelCase
обрабатывает его, согласно правилам psr-4
